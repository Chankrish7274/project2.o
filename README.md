# Bench Allocation Management System

A full-stack MERN application for managing office seating allocations.

## Features
- **Role-Based Access**: CEO, Manager, Employee.
- **Bench Management**: Create, Assign, Deallocate benches.
- **Request Workflows**: Employees can request seats; Managers approve/reject.
- **Visual Status**: Color-coded bench availability.

## Tech Stack
- **Frontend**: React, TailwindCSS, Vite
- **Backend**: Node.js, Express, Mongoose
- **Database**: MongoDB

## Quick Start
1. Start Backend: `cd server && npm run dev`
2. Start Frontend: `cd client && npm run dev`
3. Visit `http://localhost:5173`
